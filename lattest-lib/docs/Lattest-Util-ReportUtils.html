writeResults :: FilePath -> [TestResult] -> Seq TestResult -> Int -> IO (Seq TestResult) Source #

Append a list of TestResult rows to a CSV file, with optional buffering given by threshold. - If threshold is 0, results are written immediately. - If threshold > 0, results are buffered until the buffer length reaches the threshold, at which point they are written to the file. Returns the updated (reverseBuffer, bufferLength) pair. Note: this function assumes the CSV file already exists.

flushResults :: FilePath -> Seq TestResult -> IO () Source #

Flush remaining TestResult rows to a CSV file. Note: this function assumes the CSV file already exists.

initResultsFile :: FilePath -> Maybe (String, String, String) -> IO () Source #

Initialize a CSV file for test results with an optional header. If no header is provided, a default one will be used.
