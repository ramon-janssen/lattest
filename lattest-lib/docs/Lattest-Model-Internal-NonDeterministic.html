Lattest.Model.Internal.NonDeterministic

Description

Non-deterministic model definitions, where observable behaviour may lead to a set of states. These are equivalent to using Lattices with only (\/), and defined in this separate module because (/\) is more often the intended meaning.

data NonDet q Source #

Non-deterministic state configuration. This means that an automaton non-deterministically in a number of states, where zero states indicates the forbidden configuration, or in an explicit underspecified configuration.
