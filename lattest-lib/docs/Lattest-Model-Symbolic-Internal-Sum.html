Sum types

type FreeSum a = FreeMonoidX (SumTerm a) Source #

FreeSum represents a symbolic sum of terms of the type parameter a. The same term can occur multiple times.

newtype SumTerm a Source #

Terms of a free-monoids of the form:

a0 <> a1 <> ... <> an-1

where <> will be interpreted as the arithmetic sum of terms:

a0 + a1 + ... + an-1
