Product type

type FreeProduct a = FreeMonoidX (ProductTerm a) Source #

FreeProduct represents a symbolic product of terms of the type parameter a. The same term can occur multiple times.

newtype ProductTerm a Source #

Terms of a free-monoids of the form:

a0 <> a1 <> ... <> an-1

where <> will be interpreted as the arithmetic multiplication of terms:

a0 * a1 * ... * an-1
